PROJECT NAME:
WEBSCRAPPING a COVID-19 WEBPAGE FOR DATA ANALYSIS

DESCRIPTION:
In this project, I extracted data from a webpage 'https://en.wikipedia.org/w/index.php?title=Template:COVID-19_testing_by_country', and performed several data analytic processes on it to gain insights about the data.

Tasks performed:
1) Get a COVID-19 pandemic Wiki page using HTTP request
2) Extract COVID-19 testing data table from the wiki HTML page
3) Pre-process and export the extracted data frame
4) Get a subset of the extracted data frame
5) Calculate worldwide COVID testing positive ratio
6) Identify country names with a specific pattern
7) Pick two countries you are interested in, and then review their testing data
8) Compare which one of the selected countries has a larger ratio of confirmed cases to population
9) Find countries with confirmedcases to population ratio rate less than 1%
